■ PolyPeople Series - Zombies

Thank you for downloading this asset!

---

■ Package Contents

- Meshes: 16 FBX files
  - 8 with armature (rigged)
  - 8 without armature (non-rigged)
  - Includes Adult, Elder, Kid, and Teen (Female & Male)

*If you need a model without an armature, for example when using auto-rigging tools, please use the FBX files in the NoArmature folder.
For instructions on applying Mixamo animations, see Documentation/MixamoUsageGuide.txt.


- Materials: 4 total
  - Standard: 1  
  - Custom (Shader Graph): 3

*For detailed instructions on each material, please refer to Documentation/Material_SetupGuide.txt.


- Prefabs: 32 total
  - 4 prefab variants per character (based on material type)
  - All configured with Humanoid rig

- Textures:
  - 3 Base color textures (512x512) providing different color variations.


- Scenes:
  - 1 demo scene showcasing all prefabs

---

■ Geometry

All character models are under 600 triangles. 

---

■ Requirements

- Unity Version: Unity 6 (Tested with 6000.3.8f1) or higher.  
- Render Pipeline: Universal Render Pipeline (URP) required for Shader Graph materials.
