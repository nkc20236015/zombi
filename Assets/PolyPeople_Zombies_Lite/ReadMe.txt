■ PolyPeople Series - Zombies [Lite Version]

Thank you for downloading the PolyPeople Series - Zombies (Lite)!

This Lite pack is a trial version of the full "PolyPeople Series - Zombies" asset, featuring one selected character from the complete collection.

---

■ Package Contents

- Meshes: 2 FBX files (Female_Kid)
  - 1 with armature / rigged
  - 1 without armature

*If you need a model without an armature, for example when using auto-rigging tools, please use the FBX files in the NoArmature folder.
For instructions on applying Mixamo animations, see Documentation/MixamoUsageGuide.txt.


- Materials: 4 total
  - Standard: 1  
  - Custom (Shader Graph): 3

*For detailed instructions on each material, please refer to Documentation/Material_SetupGuide.txt.


- Prefabs:
  - 4 prefabs (1 character x 4 material variations)
  - All configured with Humanoid rig

- Textures: 3 base color textures (512x512)
  - Note: This texture is shared with the full version of the asset, and includes unused areas for other characters.

- Scenes:
  - 1 demo scene showcasing the character prefabs

---

■ Geometry

Character model is 582 triangles.

---

■ Requirements

- Unity Version: Unity 6 (Tested with 6000.3.8f1) or higher.  
- Render Pipeline: Universal Render Pipeline (URP) required for Shader Graph materials.
